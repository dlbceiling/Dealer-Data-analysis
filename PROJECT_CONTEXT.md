# 经销商数据分析助手业务规则上下文

更新时间：2026-06-05

本文档用于沉淀当前项目所有已确认业务规则。后续开发、修改、验收时，以本文档为业务口径参考。

## 1. 软件定位

- 本软件是 Windows 本地桌面工具。
- 使用对象为公司内部人员。
- 主要用途：
  - 分析单个经销商的 Excel 出货数据。
  - 统计销售额、产品排行、分类占比、蜂窝板面积、电器销量。
  - 结合 SKU 成本清单计算经销商实际毛利额、毛利率。
  - 导出分析结果 Excel 和 PDF 报告。
- 软件界面为简体中文。
- 一次只分析一个经销商。

## 2. 技术栈规则

- Python 3.14。
- PySide6：桌面 GUI。
- Pandas：Excel 数据处理。
- openpyxl：Excel 读取与导出。
- Plotly：图表生成。
- Qt WebEngine：显示 Plotly 图表与导出 PDF。
- PyInstaller：打包 exe。

## 3. 经销商出货 Excel 输入规则

### 3.1 文件规则

- Excel 第一行必须是字段名。
- 固定格式。
- 只分析一个经销商。
- 其他未使用字段自动忽略。

### 3.2 必要字段

经销商出货 Excel 必须包含以下字段：

```text
经销商名称
单据日期
营销分类
商品名称
单位
已发数量
单价
金额
尺寸
```

### 3.3 数据清洗规则

- `商品名称` 为空的行不参与分析。
- `经销商名称`、`营销分类`、`商品名称`、`单位`、`尺寸` 去除前后空格。
- `单据日期` 转 datetime。
- `已发数量`、`单价`、`金额` 转数值，无法转换时按 0。
- 金额为负数时正常参与统计。
- 基于 `单据日期` 生成：
  - `日期`
  - `周`
  - `月份`
  - `年份`
  - `月份序号`

## 4. 统计周期规则

- Dashboard 顶部显示：

```text
经销商：xxx
统计周期：yyyy-mm-dd ～ yyyy-mm-dd（共N条）
```

- 最小日期 = `单据日期` 最早日期。
- 最大日期 = `单据日期` 最晚日期。
- 条数 = 当前 Excel 原始总条数。
- 即使后续有被忽略的分析行，统计周期条数仍使用原始 Excel 行数。

## 5. 商品汇总规则

### 5.1 商品汇总

- `商品汇总` 按原始 `商品名称` 精确合并。
- 汇总字段：
  - `已发数量` 求和。
  - `金额` 求和。

### 5.2 明细页展示规则

- 明细页不展示原始流水。
- 明细页展示筛选后的商品汇总。
- 汇总条件：原始 `商品名称` 相同。
- 支持：
  - 搜索商品。
  - 日期筛选。
  - 营销分类筛选。
  - 表格排序。
- 筛选逻辑：
  - 先对原始清洗后明细做筛选。
  - 再按 `商品名称` 合并展示。
- 明细页汇总展示字段包括：
  - 商品名称
  - 营销分类
  - 单位
  - 已发数量
  - 金额
  - 面积㎡
  - 成本金额
  - 毛利额
  - 实际毛利率
  - 单位成本
  - 原定毛利率
  - 成本状态
  - 首单日期
  - 末单日期

## 6. 商品金额排行规则

- 商品金额 TOP3 和商品金额 TOP10 使用同一归并逻辑。
- 不直接按完整商品名称统计。
- 先生成“排行商品名称”：
  - 遇到中文括号 `（` 截断。
  - 遇到英文括号 `(` 截断。
  - 去掉左右空格。
- 括号内的工艺、颜色备注合并为同一产品。

示例：

```text
DB306-FHZ-A8L(白) -> DB306-FHZ-A8L
D-FW48G-4 (封边白) -> D-FW48G-4
D-FW48G-4 (封边黑) -> D-FW48G-4
```

- 汇总字段：
  - `已发数量` 求和。
  - `金额` 求和。
- 排序规则：
  - 按 `金额` 降序。
- 已取消商品销量 TOP10 图表。

## 7. 营销分类销售额规则

- 营销分类销售额按销售表原始 `营销分类` 统计。
- 汇总字段：
  - `已发数量` 求和。
  - `金额` 求和。
- 营销分类销售额占比图使用该汇总。

## 8. 电器/取暖器销量规则

### 8.1 电器/取暖器分类识别

以下营销分类定义为电器/取暖器：

```text
A
X
Z
T
S
J
G
D+
C
```

同时兼容以下形式：

```text
A系列
X系列
Z系列
T系列
S系列
J系列
G系列
C系列
```

额外计入电器/取暖器：

```text
凉霸
```

分类清洗规则：

- 去除前后空格。
- 去除普通空格。
- 去除全角空格。
- `D＋` 视为 `D+`。

### 8.2 电器型号归并

- 电器销量 TOP3 按型号归并。
- 型号提取规则：
  - 商品名称遇到中文括号 `（` 截断。
  - 商品名称遇到英文括号 `(` 截断。
  - 去掉左右空格。

示例：

```text
DB306-FHZ-A8L(白) -> DB306-FHZ-A8L
DB306-FHZ-A8L(灰) -> DB306-FHZ-A8L
DB100-FH-S1（白） -> DB100-FH-S1
DB100-FH-S1（灰） -> DB100-FH-S1
```

### 8.3 JS2 特殊归并

以下产品都归并为 `JS2`：

```text
DB100-FH-JS2（白）
DB100-FH-JS2（灰）
DB100-FH-JS2（米家款）白
DB100-FS-JS2(白)
```

注意：

- `延长件` 中的 `JS2-01 平面延长型材` 不计入电器销量。

### 8.4 电器排行规则

- 汇总字段：
  - `已发数量` 求和。
  - `金额` 求和。
- 排序：
  - 按合并后的 `已发数量` 降序。
- Dashboard 顶部卡片和下方表格必须使用同一份电器 TOP3 数据。

## 9. 蜂窝板/大板面积规则

### 9.1 蜂窝板分类

以下营销分类定义为蜂窝板/大板：

```text
7mm大板
9mm大板
```

### 9.2 出货面积计算

- 使用销售表 `尺寸` 字段计算实际出货面积。
- 只解析：

```text
数字*数字
```

示例：

```text
2000*1000
```

- 面积公式：

```text
面积㎡ = 长 × 宽 ÷ 1,000,000 × 已发数量
```

- 其他尺寸格式忽略，面积按 0。

### 9.3 蜂窝板统计

- 按原始 `商品名称` 汇总。
- 汇总字段：
  - 总面积㎡。
  - 总金额。

### 9.4 尺寸统计

- 仅针对蜂窝板。
- 对可解析尺寸按 `尺寸` 汇总。
- 展示面积㎡和金额。

### 9.5 7mm 标准板忽略规则

如果满足：

```text
营销分类 = 7mm大板
商品名称包含 标准板 或 标准版
```

则该行直接忽略：

- 不参与销售额。
- 不参与面积。
- 不参与成本。
- 不参与毛利。
- 不参与图表。
- 不参与排行。
- 导出到 `已忽略数据`。

业务原因：

- 该类产品已经淘汰，不需要收录和分析。

## 10. 成本清单规则

### 10.1 成本清单自动调用

- 软件启动时自动寻找成本清单。
- 优先使用上次选择过的成本文件。
- 如果无历史记录，则查找：
  - 桌面
  - 文档目录
  - 当前项目目录
- 自动识别文件名：

```text
全sku成本明细.xlsx
全SKU成本明细.xlsx
```

- 界面保留“选择成本清单”按钮，用于手动更换成本文件。

### 10.2 成本清单必要字段

成本清单必须包含：

```text
商品名称
营销类别
成本
毛利率（%）
规格
```

### 10.3 成本表清洗

- `商品名称` 去除前后空格。
- `营销类别` 去除前后空格。
- `规格` 去除前后空格。
- `成本` 转数值。
- `毛利率（%）` 转数值。
- 商品名称为空、成本为空、毛利率为空的成本行不作为有效成本。

## 11. 非大板成本规则

- 非大板按 `商品名称` 精确匹配成本表。
- 成本金额公式：

```text
成本金额 = 已发数量 × 单位成本
```

- 毛利额公式：

```text
毛利额 = 销售金额 - 成本金额
```

- 实际毛利率公式：

```text
实际毛利率 = 毛利额 ÷ 销售金额
```

- 负数销售金额正常参与统计。

## 12. 大板成本规则

### 12.1 大板成本匹配

- 大板仍优先按 `商品名称` 匹配成本表。
- 已确认别名：

```text
D-FW48G (封边白) -> D-FW48G-4 (封边白)
D-FW48G (封边黑) -> D-FW48G-4 (封边黑)
```

### 12.2 成本表规格为 custom

当成本表 `规格` 为：

```text
custom
```

表示该成本已经是单平方米成本。

公式：

```text
大板单㎡成本 = 成本
```

### 12.3 成本表规格为两段式

示例：

```text
1220×2440
1220*2440
1220x2440
```

含义：

```text
1220mm × 2440mm
```

公式：

```text
规格面积㎡ = 1220 × 2440 ÷ 1,000,000
大板单㎡成本 = 成本 ÷ 规格面积㎡
```

### 12.4 成本表规格为三段式

示例：

```text
7×600×1200
```

含义：

```text
厚度 × 宽 × 长
```

面积只取后两段：

```text
600 × 1200
```

公式：

```text
规格面积㎡ = 600 × 1200 ÷ 1,000,000
大板单㎡成本 = 成本 ÷ 规格面积㎡
```

### 12.5 大板销售成本

经销商实际大板成本必须使用销售表实际出货面积。

公式：

```text
大板销售成本 = 销售表出货面积㎡ × 大板单㎡成本
```

注意：

- 成本表规格只用于换算单㎡成本。
- 不能用成本表规格面积代替销售表出货面积。

### 12.6 整板（修边）规则

- 在 `7mm大板` 和 `9mm大板` 中，商品名称尾缀带 `（修边）` 或 `(修边)` 的数据，视为“整板”产品。
- 整板产品与同型号但名称中不带 `（修边）` 的商品不是同一成本口径，需要单独录入整板成本。
- 整板成本仍按成本表 `规格` 先换算面积，再用 `成本 ÷ 规格面积㎡` 得到单㎡成本。
- `D-FW48G (修边)` 视同 `D-FW48G-04 (修边)`。
- 整板销售面积按销售表 `尺寸` 计算，支持：
  - `1220×2440`
  - `1220×3040`
  - `7×1220×3600`
  - `9×1220×3040`
- 整板成本匹配时，需按 `商品名称 + 规格` 精确匹配，不再仅按商品名称唯一匹配。
- 若整板销售规格为 `1220×3600` 或 `1220×3000`，且成本表中没有该规格的单独整板成本，则暂按 `1220×2440` 整板成本作为兜底口径。
- 当前阶段暂不处理营销分类名称后缀为 `实芯板` 的数据，先只解决整板（修边）数据。

### 12.7 特殊成本冲突处理

- `D-FW06G-KY-9 (不修边)` 若在成本表中出现多条不同成本记录，不按普通“成本冲突”处理。
- 该商品统一取单㎡成本更高的一条作为真实成本。

## 13. 成本缺失与成本异常规则

### 13.1 成本缺失

如果销售商品在成本表中找不到：

- 不按 0 成本计算。
- 毛利不计算。
- 销售额仍计入总体销售额展示。
- Dashboard 提示：
  - 缺失行数。
  - 缺失金额。
- 导出到 `成本缺失清单`。

特殊规则：

- 若 `营销分类 = 售后配件` 且成本表中找不到成本：
  - 不再按“成本缺失”处理。
  - 实际成本按 `出货单价 × 已发数量 × 90%` 计算。
  - 单位成本按 `出货单价 × 90%` 计算。
  - 毛利额与实际毛利率按该估算成本继续计算。
  - 成本状态标记为 `售后配件估算成本`。

### 13.2 成本冲突

如果成本表中同一 `商品名称` 出现多条有效成本记录：

- 若单位成本、单㎡成本、原定毛利率一致：
  - 自动去重。
- 若单位成本、单㎡成本、原定毛利率不一致：
  - 判定为成本冲突。
  - 该商品毛利不计算。
  - 导出到 `成本异常清单`。

### 13.3 大板规格异常

如果大板成本表规格无法解析：

- 该商品毛利不计算。
- 标记为 `大板规格异常`。
- 导出到 `成本异常清单`。

## 14. 总体毛利规则

### 14.1 实际总体毛利额

```text
实际总体毛利额 = 所有已成功计算成本行的毛利额求和
```

### 14.2 实际总体毛利率

```text
实际总体毛利率 = 实际总体毛利额 ÷ 当前分析销售总额
```

说明：

- 分母包含当前分析销售总额。
- 分子只包含已成功计算成本的毛利额。
- 成本缺失和成本异常会单独提示。
- 该口径用于避免把缺失成本当成 0 成本导致毛利虚高。

## 15. 营销分类毛利率排行规则

- 按销售表原始 `营销分类` 统计。
- 统计字段：
  - 销售金额
  - 成本金额
  - 毛利额
  - 毛利率
- 毛利率公式：

```text
毛利率 = 毛利额 ÷ 销售金额
```

- 排序：
  - 按毛利率降序。

### 15.1 排除排行的分类

以下分类不参与“营销分类毛利率排行”：

```text
包装费用
促销礼品
```

注意：

- 这两个分类仍参与总销售额和总体毛利计算。
- 只是从毛利率排行表中排除。

## 16. Dashboard 规则

Dashboard 顶部展示：

- 经销商名称。
- 统计周期。
- 本期总出货金额。
- 按金额 TOP3 产品。
- 电器销量 TOP3。
- 蜂窝板总面积。
- 实际总体毛利额。
- 实际总体毛利率。
- 成本缺失提示。

Dashboard 下方展示：

- 按金额 TOP3 产品表。
- 电器销量 TOP3 表。
- 蜂窝板统计表。
- 营销分类毛利率排行表。
- Plotly 图表区。

## 17. 图表规则

### 17.1 月度进货额趋势

- 图表标题：月度进货额趋势。
- 横轴固定显示 1-12 月。
- 不同年份用不同颜色折线。
- 方便跨年份对比同月份进货趋势。

### 17.2 营销分类销售额占比

- 饼图。
- 按原始 `营销分类` 汇总金额。

### 17.3 商品金额 TOP10

- 柱状图。
- 使用“排行商品名称”归并逻辑。
- 按金额降序取 TOP10。

### 17.4 尺寸统计

- 柱状图。
- 仅针对蜂窝板。
- 按尺寸汇总面积。

### 17.5 已取消图表

- 商品销量 TOP10 已取消。

## 18. 导出规则

导出 Excel 包含以下 Sheet：

```text
清洗后明细
商品汇总
营销分类汇总
蜂窝板统计
营销分类毛利
成本缺失清单
成本异常清单
已忽略数据
```

### 18.1 清洗后明细

- 导出原始清洗后的明细行。
- 包含成本和毛利字段。
- 不包含已忽略的 7mm 大板标准板/标准版。

### 18.2 商品汇总

- 按原始商品名称汇总。

### 18.3 营销分类汇总

- 按原始营销分类汇总。

### 18.4 蜂窝板统计

- 按商品名称汇总面积和金额。

### 18.5 营销分类毛利

- 按营销分类展示销售金额、成本金额、毛利额、毛利率。
- 不包含 `包装费用` 和 `促销礼品` 的排行数据。

### 18.6 成本缺失清单

- 展示成本缺失商品。
- 用于后续补全成本表。

### 18.7 成本异常清单

- 展示成本冲突或大板规格异常商品。

### 18.8 已忽略数据

- 展示被规则剔除的数据。
- 当前主要为 `7mm大板` 中 `标准板/标准版` 商品。

## 19. 打包与迁移规则

- 使用 `build_exe.bat` 打包。
- 打包后需要复制整个目录：

```text
dist/经销商数据分析助手/
```

- 不能只复制 exe 文件。
- 另一台电脑运行 exe 不需要安装 Python。
- 若要继续开发，需要复制源码目录或使用 Git，不应基于 `dist` 目录开发。

## 20. 当前已知业务风险

- 商品金额排行的括号截断规则可能会误合并某些括号内不是颜色/工艺的真实差异产品。
- 成本别名目前写在代码中，不方便业务人员维护。
- 成本清单如果有多版本成本，当前不支持按日期选择成本版本。
- 成本缺失商品虽然进入提示，但未提供一键补全成本的流程。
- 明细页已经变成汇总视图，如需要追溯原始流水，后续应增加“原始流水”页签。
