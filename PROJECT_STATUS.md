  - 蜂窝板总面积。
  - 实际总体毛利额。
  - 实际总体毛利率。
  - 成本缺失/异常提示。

  - 月度进货额趋势，横轴固定 1-12 月，跨年份用不同颜色折线。

## 3. 已实现业务规则

### 输入 Excel 规则

- 出货 Excel 第一行必须是字段名。
- 必要字段：
  - `经销商名称`
  - `单据日期`
  - `营销分类`
  - `商品名称`
  - `单位`
  - `已发数量`
  - `单价`
  - `金额`
  - `尺寸`
- 其他字段忽略。
- 一次分析一个经销商。
- 若 `经销商名称` 去空格后的唯一值数量大于 1，则阻止分析并提示用户先导出单个经销商数据。

### 基础清洗规则

- `商品名称` 为空的行不参与分析。
- `单据日期` 转为 datetime。
- 生成：
  - `日期`
  - `周`
  - `月份`
  - `年份`
  - `月份序号`
- `金额`、`已发数量`、`单价` 转数值，无法转换时按 0。
- 负数金额正常参与统计。

### 商品汇总规则

- `商品汇总` 按原始 `商品名称` 精确合并。
- 明细页按原始 `商品名称` 合并同类项后展示。
- 商品金额 TOP3/TOP10 使用“排行商品名称”：
  - 遇到中文括号 `（` 或英文括号 `(` 截断。
  - 括号内工艺、颜色备注合并为同一个产品后排名。

### 取暖器/电器销量规则

- 电器/取暖器分类识别：
  - `A`
  - `X`
  - `Z`
  - `T`
  - `S`
  - `J`
  - `G`
  - `D+`
  - `C`
  - 以上分类的 `A系列/J系列/...` 形式
  - `凉霸`
- 兼容：
  - 全角加号 `D＋`
  - 分类前后空格
- 电器型号归并：
  - 商品名遇到 `（` 或 `(` 截断。
  - `DB100-FH-JS2（白）`、`DB100-FH-JS2（灰）`、`DB100-FH-JS2（米家款）白` 归并为 `JS2`。
  - `DB100-FS-JS2(白)` 也归并为 `JS2`。
  - 其他型号如 `DB306-FHZ-A8L(白)` 归并为 `DB306-FHZ-A8L`。
- 电器 TOP3 按合并后的 `已发数量` 降序排名。

### 蜂窝板/大板面积规则

- 蜂窝板分类：
  - `7mm大板`
  - `9mm大板`
- 出货面积按销售表 `尺寸` 计算：
  - 只解析 `数字*数字`。
  - 面积㎡ = 长 × 宽 ÷ 1,000,000 × 已发数量。
- `7mm大板` 中商品名称包含 `标准板` 或 `标准版` 的记录：
  - 直接剔除。
  - 不参与销售、面积、毛利、图表、排行。
  - 导出到 `已忽略数据`。

### 成本与毛利规则

- 成本清单必要字段：
  - `商品名称`
  - `营销类别`
  - `成本`
  - `毛利率（%）`
  - `规格`
- 非大板成本：
  - 按 `商品名称` 精确匹配。
  - 成本金额 = 已发数量 × 单位成本。
- 大板成本：
  - 按 `商品名称` 匹配成本表。
  - `规格 = custom` 时，成本即单平方米成本。
  - `规格 = 1220×2440`、`1220*2440`、`1220x2440` 时：
    - 面积㎡ = 1220 × 2440 ÷ 1,000,000。
    - 单㎡成本 = 成本 ÷ 面积㎡。
  - `规格 = 7×600×1200` 这类三段式时：
    - 按后两段 `600×1200` 算面积。
    - 单㎡成本 = 成本 ÷ 面积㎡。
  - 经销商大板成本 = 销售表出货面积㎡ × 单㎡成本。
- 整板（修边）规则：
  - 在 `7mm大板` 和 `9mm大板` 中，商品名称尾缀带 `（修边）` 的数据视为整板产品。
  - 整板成本需要单独录入，且与同型号不带 `（修边）` 的商品分开。
  - 整板成本按 `商品名称 + 规格` 匹配。
  - 销售表 `尺寸` 现支持 `×` 和三段式尺寸，用于整板面积换算。
  - 当前阶段暂不处理营销分类后缀为 `实芯板` 的数据。
- 大板别名映射：
  - `D-FW48G (封边白)` → `D-FW48G-4 (封边白)`
  - `D-FW48G (封边黑)` → `D-FW48G-4 (封边黑)`
- 毛利公式：
  - 毛利额 = 销售金额 - 成本金额。
  - 毛利率 = 毛利额 ÷ 销售金额。
  - 负数金额正常参与统计。
- 成本缺失：
  - 不按 0 成本计算。
  - 毛利显示为不可计算。
  - 销售额仍计入总体销售额展示。
  - Dashboard 提示缺失行数和金额。
  - 导出到 `成本缺失清单`。
- 售后配件缺成本特殊规则：
  - 若 `营销分类 = 售后配件` 且未匹配到成本，则按 `单价 × 已发数量 × 90%` 记作实际成本。
  - 单位成本按 `单价 × 90%` 计算。
  - 成本状态标记为 `售后配件估算成本`。
  - 该类记录不进入 `成本缺失清单`。
- 明细页商品汇总：
  - 若汇总后没有任何一行成功计算成本，则成本金额/毛利额/实际毛利率显示为“不可计算”。
  - 若部分行已成功计算、部分行成本缺失或异常，则仅汇总已成功计算部分，并通过 `成本状态` 标记“部分成本缺失”或“部分成本异常”。
- 成本冲突：
  - 同一商品名成本/单㎡成本/原定毛利率不一致时视为冲突。
  - 该商品毛利不计算。
  - 导出到 `成本异常清单`。
- 总体毛利率：
  - 分母为当前分析销售总额。
  - 分子为已匹配成本且已计算的毛利额。
  - 同时提示成本缺失/异常金额。
- 营销分类毛利率排行：
  - 按销售表 `营销分类` 统计。
  - 排除 `包装费用` 和 `促销礼品` 的毛利率排行。
  - 这两个分类仍保留在总销售额和总毛利计算中。

### 图表规则

- 月度进货额趋势：
  - 横轴为 1-12 月。
  - 不同年份使用不同颜色折线。
- 已取消商品销量 TOP10 图表。
